# Enterprise CI/CD Pipeline for Loan Management System

## Project Overview

This project demonstrates an end-to-end Enterprise DevOps CI/CD Pipeline using a Spring Boot Loan Management System deployed on AWS.

The objective is to automate the complete software delivery lifecycle from source code management to Kubernetes deployment.
